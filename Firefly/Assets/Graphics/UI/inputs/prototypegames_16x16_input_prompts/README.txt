16x16 Input Prompts
====================

Author: PrototypeGames
Version: 1.0
License: CC0 1.0 Universal (Public Domain)

Description:
------------
This is pixel art input prompts asset pack. Each input prompt is 16 pixel wide and 16 pixels tall.

Contents:
---------
- 1 tileset (277 tiles)
- Input prompts for both keyboard and controller users

How to Use:
-----------
You can import the PNG files into any game engine like Godot, Unity, or GameMaker.
